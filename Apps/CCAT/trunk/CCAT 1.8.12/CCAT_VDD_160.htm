What's New for CCAT v1.6.0

CCAT v1.6.0 was released in November 2001, and addressed the remaining open SCRs and some targets of opportunity that appeared during the summer of 2001.

Functional Modifications

Archive Processing

  • Redesigned the archive processing methodology to use a Wizard approach to force the user to input all of the information necessary to process an archive. The Wizard then properly sequences the tasks to increase the chances of successfully processing an archive

Message Translation

  • To support the MUSTANG development, code was added to parse the MTSSERROR message. This message contains error and status information that proved useful in diagnosing MUSTANG discrepancies.

  • A problem was discovered while processing CCOS 2.3 archives. It was later discovered that the developer of the MTSDALARM message changed the structure without informing us about it. Also, the change was made in such a way that it prevented a simple solution. As a result, the software now needs to know the version of the archive so it can correctly process that one message.

User Interface

  • An SCR was addressed regarding the Tools menu while no database was in use. Most of the Tools menu items are related to the current database and archive, so with no database, it was causing errors. Checks were put into place to see if there is a database and that it is currently selected.

  • Updated the Help file to include more information, including the new Archive Wizard, and detailed information on how messages are translated
